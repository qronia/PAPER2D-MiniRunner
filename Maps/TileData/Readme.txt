Tiled의 오브젝트 레이어와 타일 레이어 둘다 이용하지만, 이름은 상관 없습니다. 하지만 맵을 만드실때 동봉된 맵 타일셋을 사용하셔야합니다. 이것은 해당 맵 타일셋으로부터 이미지 정보를 가져와 분석하는 것이 아닌 해당 타일들의 ID를 매핑 정보로서 사용되기 때문입니다. 또한 타일셋의 경로는 상관 없으나, 이름을 변경하셔서는 안됩니다!!

열쇠는 4종류입니다: 빨강, 파랑, 초록, 그리고 마스터키.
마스터키는 모든 종류의 문을 여는데 사용될 수 있고, 각각의 색열쇠는 각자에 맞는 문만을 열 수 있습니다.
열쇠는 소모품이며, 색 열쇠가 먼저 사용됩니다.


또한 열쇠를 필요로 하는 키블럭과 플레이어 시작 위치는 오브젝트 레이어로부터 만들어집니다. 이들은 Type 프로퍼티와 Name 프로퍼티로 설정되며, 모두 "소문자"로만 작성되어야 합니다.

플레이어는 Type 프로퍼티에 player라고 입력하시면 됩니다. 그리하면 영역 내 가운데 위치에 스폰을 시도할 것입니다. 다만 플레이어 생성 위치는 단 하나만 존재해야하며, 그렇지 않을 경우 의도한 대로 만들어지지 않을 것입니다.

열쇠블럭은 Type 프로퍼티에 key라고 입력하고, Name에 부가 정보(red, blue, green, master)를 넣으시면 됩니다. 만일 아무것도 적지 않으시거나 올바르지 않은 키워드일 경우 master로 인식될 것입니다. 설정하신 영역 만큼 자리를 차지하도록 생성하게 될 것입니다.

마지막으로, 오브젝트 레이어에서 Text를 이용하실 수 있습니다. 영어만 가능하며, 사용하실 경우 Type에 text, Name에 내용을 적어주시면 됩니다. 만일 개행이 필요하실 경우, <br>을 작성해주시면 개행처리 됩니다.
